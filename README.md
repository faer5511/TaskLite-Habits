# TaskLite Habits

🎯 **TaskLite Habits** - это веб-приложение для управления задачами и трекинга привычек. Простой, красивый и функциональный менеджер задач с доской Kanban.

## ✨ Возможности

### 📝 To Do лист
- ➕ Добавление задач с описанием и приоритетом
- ✅ Отметка о выполнении
- 🔍 Поиск по задачам
- 🏷️ Фильтрация (Все / Активные / Выполненные)
- 📊 Сортировка (по дате, приоритету, названию)
- 📈 Статистика (всего / активных / выполненных задач)
- 🗑️ Очистка выполненных задач
- 💾 Автосохранение в localStorage

### 📌 Kanban доска
- 🎯 Три колонки: To Do, In Progress, Done
- 🖱️ Drag & Drop - перетаскивание задач между колонками
- ➕ Добавление задач прямо в колонку
- 🏷️ Визуальные метки приоритетов
- 📅 Отображение даты создания
- 🔄 Синхронизация с To Do листом

### 🎨 Дизайн
- 🌈 Современный градиентный фон
- 💨 Эффекты размытия (backdrop-filter)
- 📱 Адаптивный дизайн для всех устройств
- 🎭 Плавные анимации и переходы
- 🎨 Кастомный скроллбар
